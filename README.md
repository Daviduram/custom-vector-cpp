# Custom Vector Implementation in C++

A custom implementation of a dynamic vector container similar to `std::vector`.

## Features
- Dynamic resizing
- Push and pop operations
- Insert and erase
- Copy constructor
- Assignment operator
- Exception handling
- Manual memory management

## Technologies
- C++
- GNU Make
- Linux
- Valgrind

## What I Learned
This project helped me better understand:
- dynamic memory allocation
- constructors and destructors
- object lifetime
- templates
- low-level container implementation
